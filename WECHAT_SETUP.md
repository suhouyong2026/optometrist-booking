# 微信公众号配置指南

## 📋 前提条件

- **公众号类型**：微信服务号（已认证）
- **服务器**：已部署到腾讯云（或其他支持 HTTPS 的服务器）
- **域名**：已备案的域名（用于微信回调）

---

## 一、获取微信公众号配置信息

### 1.1 登录公众号后台

访问：https://mp.weixin.qq.com

使用公众号管理员微信扫码登录

### 1.2 获取 AppID 和 AppSecret

1. 点击左侧菜单：**开发** → **基本配置**
2. 记录以下信息：
   - **AppID(应用 ID)**：`wx1234567890abcdef`
   - **AppSecret(应用密钥)**：`1234567890abcdef1234567890abcdef`

⚠️ **重要**：AppSecret 只有管理员可见，请妥善保管

### 1.3 配置服务器域名

1. 点击左侧菜单：**开发** → **基本配置**
2. 找到 **服务器配置** 部分
3. 点击 **修改配置**

填写以下信息：

| 配置项 | 填写内容 |
|--------|----------|
| **URL** | `https://你的域名/api/wechat/verify` |
| **Token** | `自定义令牌（如：tjstandardglasses2024）` |
| **EncodingAESKey** | 点击"随机生成" |

4. 点击 **提交** 保存

---

## 二、配置网页授权域名

### 2.1 设置网页授权域名

1. 点击左侧菜单：**开发** → **公众号设置** → **功能设置**
2. 找到 **网页授权域名** 部分
3. 点击 **设置**

填写你的域名（不带协议）：
```
你的域名.tcloudbaseapp.com
```

或自定义域名：
```
glasses.tianjinstandard.com
```

⚠️ **注意**：
- 域名必须已备案
- 必须使用 HTTPS
- 域名需要能正常访问

### 2.2 下载验证文件

1. 点击 **下载验证文件**
2. 将下载的 `MP_verify_xxxx.txt` 文件上传到你网站的根目录
3. 确保可以通过 `https://你的域名/MP_verify_xxxx.txt` 访问
4. 点击 **确定** 完成验证

---

## 三、配置 JS 接口安全域名

### 3.1 设置安全域名

1. 点击左侧菜单：**开发** → **公众号设置** → **功能设置**
2. 找到 **JS 接口安全域名** 部分
3. 点击 **设置**

填写你的域名（不带协议）：
```
你的域名.tcloudbaseapp.com
```

⚠️ **注意**：
- JS 接口安全域名用于微信 JS-SDK 调用
- 必须与网页授权域名一致
- 同样需要域名验证

## 四、配置服务器环境变量

### 4.1 本地开发环境

在项目根目录创建 `.env` 文件：

```bash
# 微信公众号配置
WECHAT_APPID=wx1234567890abcdef
WECHAT_SECRET=1234567890abcdef1234567890abcdef

# 腾讯云配置
TENCENT_ENV_ID=suhouyong2026-5gq178it64857137

# 服务器配置
PORT=3000
NODE_ENV=development
BASE_URL=http://localhost:3000
```

### 4.2 生产环境（腾讯云）

在腾讯云云函数控制台配置环境变量：

1. 进入云函数详情页
2. 点击 **环境变量** 标签
3. 添加以下变量：

| 变量名 | 值 |
|--------|-----|
| `WECHAT_APPID` | `wx1234567890abcdef` |
| `WECHAT_SECRET` | `1234567890abcdef1234567890abcdef` |
| `TENCENT_ENV_ID` | `suhouyong2026-5gq178it64857137` |
| `BASE_URL` | `https://你的域名.tcloudbaseapp.com` |

⚠️ **重要**：
- `BASE_URL` 必须使用 HTTPS
- 微信回调地址必须是公网可访问的

## 五、配置自定义菜单（可选）

### 5.1 创建自定义菜单

1. 点击左侧菜单：**内容与互动** → **自定义菜单**
2. 点击 **添加菜单**

创建菜单结构：

```
├─ 预约服务
│  ├─ 在线预约（跳转网页）
│  ├─ 我的预约（跳转网页）
│  └─ 门店信息（跳转网页）
├─ 会员中心
│  ├─ 会员权益（跳转网页）
│  └─ 积分查询（跳转网页）
└─ 联系客服
   ├─ 在线咨询（发送消息）
   └─ 门店导航（发送位置）
```

### 5.2 配置菜单链接

**在线预约** 菜单：
- 菜单类型：跳转网页
- 网页地址：`https://你的域名.tcloudbaseapp.com/`

**我的预约** 菜单：
- 菜单类型：跳转网页
- 网页地址：`https://你的域名.tcloudbaseapp.com/profile.html`

**员工入口** 菜单（可选）：
- 菜单类型：跳转网页
- 网页地址：`https://你的域名.tcloudbaseapp.com/staff.html`

### 5.3 发布菜单

1. 配置完成后点击 **保存并发布**
2. 菜单会在 24 小时内生效（通常几分钟即可）
3. 用户需要重新关注公众号或清除缓存才能看到新菜单

## 六、测试验证

### 6.1 测试微信登录

1. 在微信内打开公众号
2. 点击自定义菜单"在线预约"
3. 点击"微信一键登录"按钮
4. 应该跳转到微信授权页面
5. 授权后自动返回，显示用户头像和昵称

### 6.2 检查授权回调

访问以下 URL 测试微信授权（替换 AppID 和域名）：

```
https://open.weixin.qq.com/connect/oauth2/authorize?appid=YOUR_APPID&redirect_uri=https://你的域名/api/wechat/callback&response_type=code&scope=snsapi_userinfo&state=STATE#wechat_redirect
```

如果正确跳转到授权页面，说明配置成功

### 6.3 查看日志

在服务器查看日志，确认授权流程：

```bash
# 本地开发
npm start

# 查看腾讯云日志
进入云函数控制台 → 日志查询
```

## 七、常见问题

### Q1: 授权失败，提示 "redirect_uri 参数错误"

**原因**：网页授权域名未正确配置

**解决**：
1. 检查网页授权域名是否已设置
2. 确认域名与 redirect_uri 一致
3. 确认域名已通过验证

### Q2: 获取用户信息失败

**原因**：AppSecret 错误或已过期

**解决**：
1. 重新获取 AppSecret
2. 检查环境变量是否正确配置
3. 确认服务器时间是否准确

### Q3: JS-SDK 无法使用

**原因**：JS 接口安全域名未配置

**解决**：
1. 配置 JS 接口安全域名
2. 确保后端正确生成签名
3. 检查 wx.config 配置是否正确

### Q4: 域名未备案

**原因**：中国大陆服务器要求域名备案

**解决**：
1. 到域名服务商处申请备案
2. 或使用已备案的域名
3. 或使用腾讯云开发域名（已备案）

## 八、安全建议

### 8.1 保护 AppSecret

- ❌ 不要将 AppSecret 提交到 Git
- ❌ 不要在前端代码中使用 AppSecret
- ✅ 使用环境变量存储
- ✅ 定期更换 AppSecret

### 8.2 访问令牌管理

- access_token 有效期 2 小时
- 需要定时刷新并缓存
- 不要每次请求都重新获取

### 8.3 用户数据安全

- 用户 openid 不要暴露给前端
- 敏感操作需要二次验证
- 定期备份数据库
