# 验光师预约系统 - 部署说明

## 一、系统说明

这是一个为天津标准眼镜和平路总店定制的验光师预约系统，专为许晓龙验光师设计。

### 系统特点
- ✅ 顾客通过微信公众号预约验光
- ✅ 许晓龙验光师可设置上班班次
- ✅ 自动生成二维码和唯一流水号
- ✅ 支持扫码核销验证
- ✅ 完整的数据统计后台

### 三个后台账号

| 角色 | 用户名 | 密码 | 功能 |
|------|--------|------|------|
| 员工 | staff | staff123 | 查看预约情况 |
| 许晓龙 | xuxiaolong | xxl2024 | 设置班次 + 核销预约 |
| 管理员 | admin | admin2024 | 查看统计报表 |

## 三、部署方案

### 方案一：Vercel 免费部署（推荐测试用）

**优点**：
- 完全免费
- 自动 HTTPS
- 全球 CDN 加速
- 无需运维

**缺点**：
- 国内访问速度一般
- 需要绑定 GitHub

**部署步骤**：

1. 将代码上传到 GitHub 仓库

2. 访问 https://vercel.com 注册账号

3. 点击 "New Project"，导入 GitHub 仓库

4. 配置构建设置：
   - Framework Preset: Node.js
   - Build Command: `npm install`
   - Output Directory: `public`
   - Install Command: `npm install`

5. 点击 "Deploy" 完成部署

6. 获得域名类似：`your-app.vercel.app`

### 方案二：阿里云轻量应用服务器（推荐生产用）

**配置推荐**：
- CPU: 1 核
- 内存：2GB
- 硬盘：40GB SSD
- 带宽：3Mbps
- **费用：约 ¥24/月**

**部署步骤**：

1. 购买服务器
   - 访问 https://www.aliyun.com
   - 选择"轻量应用服务器"
   - 选择"Node.js"镜像

2. 配置服务器
   ```bash
   # SSH 登录服务器
   ssh root@your_server_ip
   
   # 安装 Node.js（如果镜像未预装）
   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
   sudo apt-get install -y nodejs
   
   # 上传代码（使用 SCP 或 Git）
   git clone your_repo.git
   
   # 安装依赖
   cd your_project
   npm install
   
   # 使用 PM2 管理进程
   npm install -g pm2
   pm2 start server.js --name optometrist-booking
   pm2 save
   pm2 startup
   ```

3. 配置域名（可选）
   - 在阿里云购买域名（约 ¥30/年）
   - 添加 A 记录指向服务器 IP
   - 配置 Nginx 反向代理

4. 配置 HTTPS（推荐）
   ```bash
   # 使用 Let's Encrypt 免费证书
   sudo apt install certbot python3-certbot-nginx
   sudo certbot --nginx -d yourdomain.com
   ```

## 五、系统使用说明

### 顾客端使用流程

1. 顾客点击公众号菜单"预约验光"
2. 填写个人信息（姓名、年龄、电话）
3. 选择可预约日期（显示许晓龙上班的日期）
4. 选择时间段（9:00-18:00，每小时一个时段）
5. 提交预约，生成二维码和流水号
6. 保存二维码，按时到店

### 许晓龙验光师使用流程

**设置班次**：
1. 访问员工入口，使用个人账号登录
2. 进入"班次管理"标签
3. 选择月份，点击要上班的日期
4. 点击"保存班次"
5. 顾客端即可看到可预约日期

**核销预约**：
1. 顾客出示二维码
2. 扫描二维码或输入流水号
3. 点击"验证"
4. 系统显示验证结果（大字 + 图标）
   - ✓ 绿色：预约有效
   - ⚠️ 黄色：已核销
   - ✗ 红色：预约无效
5. 确认无误后点击"确认核销"

### 员工使用流程

1. 访问员工入口，使用员工账号登录
2. 查看预约列表
3. 可按日期、状态筛选
4. 查看统计数据

### 管理员使用流程

1. 访问员工入口，使用管理员账号登录
2. 选择统计月份
3. 查看总体统计和每日详细数据
4. 查看预约趋势图表

## 六、安全建议

### 1. 修改默认密码
首次登录后，建议修改默认密码：
- 修改 `server.js` 中的密码配置
- 重启服务器

### 2. 数据备份
定期备份数据文件：
```bash
# 备份数据库文件
cp data/database.json data/database_backup_$(date +%Y%m%d).json
```

### 3. 访问控制
- 后台入口不要公开传播
- 仅告知相关人员登录信息

## 七、常见问题

### Q1: 顾客看不到可预约日期？
A: 需要许晓龙先在后台设置上班班次

### Q2: 时间段显示灰色？
A: 该时段已约满（每时段限 2 人）或该日期无班次

### Q3: 二维码无法扫描？
A: 可以使用微信"识别图中二维码"功能，或手动输入流水号

### Q4: 数据会丢失吗？
A: 数据保存在服务器本地，建议定期备份

### Q5: 可以扩展到其他验光师吗？
A: 系统预留了扩展接口，可在 `server.js` 中添加更多验光师账号
