# 微信公众号预约系统改造计划

## 📋 项目概述
将现有验光预约系统改造成微信公众号原生网页，使用微信 JS-SDK 和网页授权，对接腾讯云数据库。

---

## 🎯 核心改造内容

### 一、后端改造（server.js）

#### 1. 添加微信 OAuth2.0 授权
- **新增接口**：`GET /api/wechat/auth` - 微信授权回调
- **新增接口**：`GET /api/wechat/userinfo` - 获取用户信息
- **新增接口**：`POST /api/wechat/phone` - 微信手机号获取
- 实现微信 OAuth2.0 网页授权流程（snsapi_userinfo）
- 使用 code 换取 access_token 和 openid
- 使用 access_token 获取用户信息（昵称、头像）

#### 2. 对接腾讯云数据库
- 安装 `tcb-admin-node` SDK
- 初始化云开发环境：`suhouyong2026-5gq178it64857137`
- 替换内存存储为云数据库操作
- 数据库集合：
  - `users` - 用户信息（openid、昵称、头像、手机号）
  - `bookings` - 预约记录
  - `shifts` - 班次管理

### 二、前端改造（index.html）

#### 1. 添加微信 JS-SDK
```html
<script src="https://res.wx.qq.com/open/js/jweixin-1.6.0.js"></script>
```

#### 2. 微信一键登录流程
- 页面加载时自动检测是否在微信内
- 如果未登录，自动跳转微信授权页
- 授权后回调，获取用户 openid、昵称、头像
- 显示用户信息，无需手动输入

#### 3. 微信手机号获取
- 使用微信 JS-SDK 的 `wx.chooseWXPay` 或 H5 手机号组件
- 或者使用微信服务号的"手机号快速验证"能力
- 获取后自动填充到表单

#### 4. 保持现有 UI 样式
- 不修改现有 CSS 样式
- 只增加微信登录 UI（用户头像、昵称显示）
- 保持预约流程不变

### 三、微信公众号后台配置

#### 1. 公众号类型要求
- **必须**：微信服务号（已认证）
- 开通微信支付（可选，用于验证）

#### 2. 网页授权域名配置
- 登录公众号后台：https://mp.weixin.qq.com
- 设置 → 公众号设置 → 功能设置
- **网页授权域名**：`suhouyong2026-5gq178it64857137-1404541376.tcloudbaseapp.com`
- **JS 接口安全域名**：同上

#### 3. 授权目录配置
- 授权回调页面地址：`https://你的域名/wechat-auth.html`
- 需要 ICP 备案

#### 4. 获取微信开发配置
- **AppID**：公众号的 AppID
- **AppSecret**：公众号的 AppSecret（保密）
- **令牌 (Token)**：用于服务器配置
- **EncodingAESKey**：消息加解密密钥

## ⚠️ 重要注意事项

### 合规要求
1. **域名备案**：云开发域名需要 ICP 备案
2. **HTTPS**：微信要求必须使用 HTTPS
3. **用户隐私**：需有隐私政策说明
4. **授权提示**：明确告知用户授权用途

### 技术要点
1. **微信内检测**：使用 `navigator.userAgent` 检测微信浏览器
2. **静默授权**：使用 snsapi_base 获取 openid（无需用户确认）
3. **手动授权**：使用 snsapi_userinfo 获取昵称头像（需用户确认）
4. **手机号获取**：需用户主动触发，无法自动获取
